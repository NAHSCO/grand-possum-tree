How to Use Stockkept
A guide to all features of your long-term storage tracker.

Getting Started

New to Stockkept? Click Start Here in the sidebar. Answer 5 quick questions about your household and goals — Stockkept will generate recommended baseline goals automatically and give you smart suggestions tailored to your situation.

Adding Items to Inventory

Click + Add Inventory in the header to open the item form. Required fields vary by type: Food and Water require an expiration date and calories (food only). Date Stored is required for all items. Water items use gallons or liters only and auto-fill the name "Water".

Tip: When editing an item, the form shows the item's current age based on the stored date.

Alerts & Notifications

Stockkept auto-generates alerts for expired items (red), items expiring within 90 days (amber), baseline shortfalls (blue), and milestone achievements (gold). Dismissing an alert from the Alerts page hides it without deleting the underlying item. You can also add custom alerts with three severity levels.

Baseline Goals

Set minimum storage targets per category. Food goals track calories; pet goals track weight; household and water goals track volume or quantity. Add a note to any goal for additional context. Click Log Item on a goal to add inventory directly for that category.

Import, Export & Reset

Data is saved automatically to your browser. Use Export JSON to back up your data to a file, and Import JSON to restore. Reset All permanently erases everything — export first.

Data Collection — What You Should Know

Here is every point where data could theoretically leave your device when using Stockkept:

GitHub Pages hosting — Stockkept is hosted on GitHub Pages (GitHub, Inc.). GitHub logs visitor IP addresses for security on every page load. This is infrastructure-level collection by GitHub, not by North American Health Sciences LLC, and is governed by the GitHub Privacy Statement.

Ko-fi / Buy Me a Can of Beans button — Clicking this button navigates to ko-fi.com. Ko-fi and Stripe will see your IP address, browser information, and any account information you provide. This only occurs on a deliberate click.

Feedback / Bug Report link — This link navigates to Google Forms. Google will see your IP address when you visit the form. This only occurs on a deliberate click.

localStorage (your inventory data) — All data you enter into Stockkept stays entirely within your browser. It is never transmitted anywhere. No server, no API, nothing leaving your device.

The short answer: the only passive data collection in the web version is GitHub's IP logging on page load. Everything else — Ko-fi, Google Forms — only activates when you deliberately click an external link. The download version has zero passive data collection of any kind.
